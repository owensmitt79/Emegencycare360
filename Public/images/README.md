# Emergencycare360 Image Assets

This folder contains all local image assets used throughout the project.

- `logo.png`: Main application brand logo (served via `/api/logo`).
- `hero-image.png`: Header background image (served via `/api/hero-image`).
- `dr-jane.png`: Image of Dr. Jane used on the About page.

*Note: These assets are auto-synchronized and copied from the local device folders when the Next.js dev server starts.*
